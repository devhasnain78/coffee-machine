A Python implementation of a coffee machine simulator using Object-Oriented Programming. This console-based application models a real coffee machine with resource management, payment processing, and drink preparation functionality.

## Features
**Drink Menu:** Espresso, Latte, and Cappuccino with custom recipes

**Payment System:** Accepts coins and calculates change  
**Interactive Console Interface:** User-friendly text-based interaction  
**Modular OOP Design:** Clean separation of concerns using classes
